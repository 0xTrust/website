# My Blog

Visit Blog: [https://tonyxu.io](https://tonyxu.io)

## Prerequisite

- [Node.js](https://nodejs.org/en/)
- [Hugo](https://gohugo.io/getting-started/quick-start/)
- [Firebase CLI](https://firebase.google.com/docs/cli/)

## Usage

Serve

```
hugo serve -D
```

Build

```
hugo
```

Deploy

```
firebase deploy
````
